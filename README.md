사용자는 자신의 노하우나 지식, 생각을 주어진 Template으로 만들 수 있다. 사용자는 간단한 메모, 자연어 대화, 질의 응답으로 이 과정을 수행할 수 있다.

Agent가 생성한 문서는 해당 문서가 언제 작성되었는지, 같이 작성한 사용자가 누군지를 기록할 수 있어야 한다.
Agent는 사용자와의 상호 작용을 통해 문서를 작성, 편집, 삭제한다.
    Agent는 Template의 특정 부분을 채울 수 없거나, 채우기에 부족한 경우 사용자에게 추가 내용을 요청할 수 있다.
    Agent는 사용자의 요청에 따라 다양한 Task를 수행한다.
        Agent는 사용자가 문서 열람을 요청할 경우 현재까지 작성된 모든 문서를 보여준다.
        Agent는 사용자가 문서 수정을 요청할 경우, 해당 문서를 보여주며 수정을 시작한다.
            수정 과정은 사용자와의 상호 작용을 진행한다.
        Agent는 사용자가 문서 삭제를 요청할 경우, 해당 문서를 보여주며 삭제를 진행한다. 단, 삭제를 진행할 것인지 다시 한 번 물어보고 확정 한다.
